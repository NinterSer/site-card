# Запуск проекта
1. Устанавливаем Node.js
2. Качаем это репозиторий, если требуется разорхивируем в отдельную папку
3. Открываем CMD (или терминал на Linux) и переходим в каталог с проектом cd **/site-card/
4. Инициализируем проект компандой npm -init
5. После инициализации запускаем скрипт командой npm run start
6. Переходим в браузере по ссылке http://localhost:8080
